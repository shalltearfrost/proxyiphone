# Сборка iPhone Proxy на Windows — пошагово

Инструкция для друга. Приложение чисто на JavaScript (нативных модулей нет),
поэтому Visual Studio / build tools **не нужны** — только Node.js.

## 1. Установить необходимое

1. **Node.js LTS** — https://nodejs.org → кнопка LTS → установить (галочки по
   умолчанию, npm ставится вместе с ним). Проверка в PowerShell:
   ```powershell
   node -v
   npm -v
   ```
2. **Apple Devices** (драйвер Apple, чтобы Windows видел iPhone и его модем) —
   установи из Microsoft Store приложение **«Apple Devices»** (или классический
   iTunes с сайта Apple). Перезагрузись, если попросит.

## 2. Скачать проект

Вариант без Git (проще):
1. Открой https://github.com/shalltearfrost/proxyiphone
2. Зелёная кнопка **Code → Download ZIP**
3. Распакуй, например, в `C:\proxyiphone`

Вариант с Git:
```powershell
git clone https://github.com/shalltearfrost/proxyiphone.git
cd proxyiphone
```

## 3. Собрать установщик

В PowerShell в папке проекта:
```powershell
cd C:\proxyiphone        # путь, куда распаковал
npm install              # скачает зависимости и Electron (пару минут)
npm run dist:win         # соберёт установщик
```

Готовые файлы появятся в папке **`dist\`**:
- `iPhone-Proxy-0.1.0-x64.exe` — установщик (nsis)
- `iPhone Proxy 0.1.0.exe` — портативная версия (можно запускать без установки)

> При первом запуске Windows SmartScreen может предупредить «Неизвестный
> издатель» (приложение не подписано). Нажми **«Подробнее → Выполнить в любом
> случае»**.

## 4. Просто запустить без сборки (если .exe не нужен)

```powershell
npm install
npm start
```

## 5. Пользоваться

1. Подключи iPhone кабелем, на телефоне: **Настройки → Режим модема → вкл**,
   нажми **«Доверять этому компьютеру»**.
2. Запусти iPhone Proxy. В карточке телефона выбери его сетевой адаптер.
3. **«Проверить IP»** — покажет внешний IP через сим-карту телефона.
4. **«Использовать как прокси»** — появится адрес `IP:порт`.
5. На клиенте пропиши HTTP-прокси `IP-ноутбука:порт` + логин/пароль из окна.

## Если что-то не так

- **Телефон не виден / нет адаптера** — не установлен драйвер Apple (шаг 1.2),
  или не нажато «Доверять» на телефоне, или выключен Режим модема.
- **Нет заряда телефона (показывает «—»)** — это норма: без бинарников
  libimobiledevice заряд не читается, но прокси работает. Чтобы добавить заряд,
  положи `.exe`/`.dll` из https://github.com/libimobiledevice-win32/imobiledevice-net/releases
  в папку `vendor\imobiledevice\win32\` перед сборкой.
- **Другое устройство в сети не подключается к прокси** — ноутбук должен быть
  одновременно в той же сети (Wi-Fi/Ethernet) и с активным Режимом модема;
  проверь, что брандмауэр Windows не блокирует приложение (разреши в сети).
