# iPhone Proxy

Десктоп-приложение (Electron): превращает подключённые по USB iPhone в
HTTP/HTTPS прокси-серверы. Трафик каждого прокси уходит через сотовую сеть
конкретного телефона. В интерфейсе видно заряд, модель, статус, адрес прокси,
скорость и трафик. Работает на **macOS** и **Windows**.

## Как это работает

1. iPhone подключается к компьютеру по **USB-кабелю** и включает **Режим модема**
   (Personal Hotspot). Телефон появляется как сетевой адаптер со своим IP
   (обычно `172.20.10.x`).
2. Для выбранного телефона поднимается HTTP/HTTPS прокси на `0.0.0.0:<порт>`.
   Исходящие соединения привязываются к IP адаптера телефона (`localAddress`),
   поэтому трафик идёт через его сим-карту.
3. Клиенты в локальной сети подключаются к `<IP-компьютера>:<порт>` с
   логином/паролем (показаны вверху окна).
4. Заряд/имя/модель телефона читаются через `libimobiledevice` (опционально —
   без него прокси всё равно работает, просто без заряда).

Несколько телефонов = несколько USB-кабелей = несколько прокси на разных портах.

## 🪟 Windows — готовый установщик (когда собран)

1. Установи **Apple Devices** из Microsoft Store (или iTunes) — нужен драйвер
   Apple, иначе Windows не увидит iPhone. Перезагрузись, если попросит.
2. Скачай `iPhone-Proxy-Setup.exe` из раздела
   **[Releases](../../releases)** этого репозитория и установи.
3. Подключи iPhone кабелем, на телефоне включи **Настройки → Режим модема**,
   нажми **«Доверять этому компьютеру»**.
4. Запусти **iPhone Proxy**. В карточке телефона выбери его сетевой адаптер,
   нажми **«Проверить IP»** (покажет внешний IP через сим-карту), затем
   **«Использовать как прокси»**.
5. На клиенте пропиши HTTP-прокси `IP-ноутбука:порт` + логин/пароль из окна.

> Заряд телефона на Windows показывается только если рядом с приложением лежат
> бинарники `libimobiledevice` (см. `vendor/imobiledevice/win32/README.md`).
> Без них прокси работает, заряд показывается как «—».

### Как получается готовый .exe

`.exe` собирается автоматически в GitHub Actions при пуше тега `vX.Y.Z`
(workflow `.github/workflows/build-windows.yml`) и попадает в Releases.
Собрать вручную на Windows-машине: `npm install && npm run dist:win`.

## Использование (обе ОС)

1. Подключи iPhone по USB, включи Режим модема, нажми «Доверять».
2. Выбери сетевой адаптер телефона в списке карточки.
3. **«Проверить IP»** — покажет внешний IP через эту сим-карту.
4. **«Использовать как прокси»** — появится адрес `IP:порт`, скорость и трафик.
5. На клиенте укажи этот HTTP-прокси + логин/пароль.

## Важно про сеть

- Чтобы отдавать прокси **другим устройствам в сети**, компьютер должен быть
  одновременно в локальной сети (Wi-Fi/Ethernet) **и** иметь активный
  USB-модем телефона. Если Режим модема «засыпает» — держи открытым его экран
  на iPhone, либо подключи компьютер к сети по Ethernet.
- Прокси слушает на `0.0.0.0` — доступен из локальной сети, поэтому
  логин/пароль обязателен. Доступ из интернета — только через отдельный проброс
  портов и на свой риск.
- Несколько телефонов по умолчанию раздают одинаковую подсеть `172.20.10.x`.
  Кнопка «Проверить IP» показывает, через какой телефон реально идёт трафик.

## Требования

- **macOS**: Node.js + libimobiledevice (`brew install node libimobiledevice`)
- **Windows**: драйвер Apple (Apple Devices / iTunes); libimobiledevice —
  опционально для показа заряда
